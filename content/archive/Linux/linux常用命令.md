# 常用命令

## 查找文件
```shell
[root@vultr hexo]# find / -name nginx 
/usr/sbin/nginx
```

## 查找历史语句
```shell
[root@vultr hexo]# history | grep nginx
   85  yum install -y git nginx
   87  ps -ef|grep nginx
```
