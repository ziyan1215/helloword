## 形参和实参
`public static void main(String[] args)`
这里的args就是参数

形参全称为“形式参数”是在定义函数名和函数体的时候使用的参数，目的是用来接收调用该函数时传递的参数。
实参全称为"实际参数"是在调用时传递给函数的参数，即传递给被调用函数的值

这里就有个引用传递和值传递的问题需要讨论了
